## Compiling

### Requirements

* JDK 17
* Bitwig Studio Version 5.1
* Bitwig Controller API 18
* Maven 3.9.5 


### Build and install

1. Follow the installation instructions for each of the above requirements.
2. You might want to install an IDE like Eclipse or Visual Studio Code
3. Run `mvn install`.
